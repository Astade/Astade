Here you get binaries for Astade for Windows.

Because there is not yet an installation program, you only can download this files, which belong to an  installation for Windows. Copy all files in any folder and start the program "Astade.exe". There are no registry entries required, nor will the program make any.

Then, you have to use your intuition,  how to use the program, because there is not a powerful description yet. :-)

Of course, this description will follow, "soon", whenever that is.

Or, you write a mail to  users mailing list. You'll get help there! 

  • Astade.exe Is the main part of the package. This is the program you should start. If required, it will start the others.
  • ResourceEdit.exe Is the program you need to modify all model elements.
  • omd.exe Is used to draw the Object Model Diagrams. But you need
  • omd.bat in addition, but you will have to adapt this one. Use it as an example only.
  • QDC.exe is the C++ code generator. QDC stands for "Quick and Dirty Coder" so do not expect it to make too beautiful code. There will follow a better coder.
  • Makefile is a makefile template. Whenever you create new components, you can copy this, as a beginning. Put it in a "Template" folder, because you will develop you own templates soon.
  • help.zip is, where the help is located. But do not expect too much from here, we're working on it!
