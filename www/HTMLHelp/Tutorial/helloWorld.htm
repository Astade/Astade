Creating a "Hello World" application.

The design

Lets do a "hello world" program. But, to make it  object oriented, make it a little bit complicate. We will program four classes: one for the application, one for the console output and one doing the "hello world". The fourth one is an interface, to describe an output in a general way, so that the class HelloWorld writes to an abstract interface.
