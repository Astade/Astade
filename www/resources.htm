The following sites provide useful tools for the Astade project.

  • http://www.wxwidgets.org/ 
    An open source C++ GUI framework to make cross-platform programming child's play. Well, almost.
  • Project: wxwidgets Dialog Designer:
    Here you get the development environment which we use for the Astade, Windows Version.
  • http://www.graphviz.org/ 
    Graphviz is open source graph visualization software.
  • http://www.spinellis.gr/sw/umlgraph/ 
    UMLGraph allows the declarative specification and drawing of UML class and sequence diagrams.
  • http://www.reumuth.com/programming/
    Lite Edit is a real nice Editor for C++ to use with Astade under Windows.
  • http://www.doxygen.org/
    Doxygen is a documentation system for C++, C, Java, Objective-C, IDL (Corba and Microsoft flavors) and to some extent PHP, C#, and D.
  • http://www.gnu.org/software/plotutils/plotutils.html
    The GNU plotutils package contains software for both programmers and technical users. Its centerpiece is libplot, a powerful C/C++ function library for exporting 2-D vector graphics in many file formats, both vector and raster.
    You can find a Windows binary of plotutils here: http://gnuwin32.sourceforge.net/packages.html
  • http://www.oose.de/downloads/uml-2-Notationsuebersicht-oose.de.pdf
    UML notation overview (in German)
  • UML Pad is a CASE tool for UML diagrams design.
    It supports Class, Sequence, State and Activity diagrams.
