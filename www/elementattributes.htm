The model elements of astade may have the following element attributes

  • Name
    the name of the model element.
  • Short description
    a description of the element, its use, in a view words.
  • Description
    a verbose description of the element. 
  • Type
    the (return) type of the element. 
  • Visibility
    can be private, protected or public.
  • Relation-type
    can be specification-usage, implementation-usage,  inheritance, association, aggregation, composition. 
  • Default
    the default (initial) value for that model element.
  • Static
    can be true or false. 
  • Virtual
    can be true or false. 

But, of course, not all model elements have all attributes.
